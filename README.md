# 🏠 114年度財產交易所得稅計算機 (舊制)

> 專為台灣納稅義務人與不動產從業人員設計的純前端計算工具。
> 依據財政部最新公布之「一百十四年度個人出售房屋之財產交易所得計算規定」實作，無需安裝任何後端環境，開箱即用。

## ✨ 核心功能特色

* 📅 **新舊制自動防呆判定**
  自動依據「取得日期」與「交易日期」計算持有期間，若符合房地合一稅（新制）條件（105年以後取得，或103年取得且持有未滿2年），系統將自動鎖定表單並跳出警告，避免誤算。
* 📊 **雙軌計算模式無縫切換**
  * **核實認定 (所得稅法第14條)**：支援「已劃分」與「未劃分」房地契約。未劃分時，系統會嚴格依循法規，先計算總損益後，再依房地現值比例拆算房屋交易所得。
  * **標準認定 (無法證明成本)**：內建 114 年度全台各縣市、鄉鎮市區最新公告比率（自動連動下拉選單）。
* 🏛️ **高價房屋 (豪宅) 門檻動態判定**
  系統會根據所選縣市自動判斷總成交價是否達高價房屋門檻（如：台北市 6,000 萬、新北市 4,000 萬等），若達標則自動套用 20% 之法定利潤率進行計算。
* 💰 **精細的必要費用扣除清單**
  提供契稅、印花稅、代書費、土地增值稅、規費、公證費、仲介費及修繕費等逐項輸入欄位，並即時自動加總。
* 📱 **現代化響應式 UI**
  採用 Tailwind CSS 構建，無論在桌機、平板或手機上皆提供流暢、直覺的操作體驗；並內建自定義的 Tooltips（提示說明），降低使用者的法規理解門檻。

## 🛠️ 技術架構

本專案採用最輕量化的純前端技術構建，將所有邏輯整合於單一檔案中：
* **HTML5**：語意化網頁架構。
* **Vanilla JavaScript (ES6+)**：處理複雜的稅務計算邏輯、DOM 操作與狀態管理。
* **Tailwind CSS (via CDN)**：快速且一致的原子化樣式設計。

## 🚀 部署與本機執行方式

### 本機直接執行
1. 將本專案的 `index.html` 下載至您的電腦。
2. 直接點擊兩下使用瀏覽器（推薦 Chrome, Edge, Safari）開啟，即可在有網路的狀態下使用完整功能。

### 透過 GitHub Pages 免費部署
1. 註冊並登入 GitHub。
2. 建立一個新的公開 Repository（儲存庫）。
3. 將 `index.html` 上傳至該儲存庫的主分支（main/master）。
4. 進入 Repository 的 `Settings` > `Pages`。
5. 在 `Source` 選擇 `Deploy from a branch`，並指定您的主分支後儲存。
6. 等待 1-3 分鐘，GitHub 將自動生成您的專屬公開網址。
